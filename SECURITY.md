# Security Policy

## Scope

`zer0lint` is in scope for:

- unsafe file writes
- accidental disclosure of local config contents
- dependency vulnerabilities introduced by this repo
- bugs that make the tool modify configs outside the requested target

Out of scope:

- extraction quality disagreements on a specific model
- mem0 or third-party service outages
- benchmark disagreements

## Reporting

Open a private security advisory on GitHub if possible. Use public issues only for non-sensitive bugs.
